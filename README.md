# 🌱 FarmKart – Your Farm. Your Market.

FarmKart is an AI-powered full-stack agriculture marketplace that connects farmers directly with buyers. The platform eliminates middlemen and provides smart farming features such as AI price prediction, disease detection, live weather updates, transport sharing, secure payments, and order tracking.

---

## 🚀 Features

- 👨‍🌾 Farmer Dashboard
- 🛒 Buyer Dashboard
- 📦 Product Management
- 📋 Order Tracking
- 💳 Secure Payments
- 🌦️ Live Weather Forecast
- 🤖 AI Price Prediction
- 🌿 Disease Detection
- 🚚 Transport Sharing
- 📍 Nearby Farmers
- 🔔 Notifications
- 📊 Profit Report
- 🔐 JWT Authentication
- 👑 Admin & Super Admin Panel

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Sai-praveena518/FarmKart.git
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend/backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python app.py
```
